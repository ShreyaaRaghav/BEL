Tell about a project?
Ans: One project I’m proud of is Contexto — a semantic word-guessing platform. Unlike Wordle-style games that rely on spelling, this evaluates guesses based on semantic similarity using cosine distance.

Q. “When would you use Redis instead of Postgres?”
Ans: Both are storing features. Redis is in-momory storage, cache and msg broker. Postgres is DBS and database oriented. So for fast, flexible, session-based storage? Redis is more optimal as data is in memory, hence more configurable and high speed. Postgres is used when the demand is of durablility and data integrity…as it writes data into disk directly.
